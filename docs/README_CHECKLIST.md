## 🎨 Image Quality Tips

### ✅ Good Screenshots
- Real content (not empty states)
- Clean UI (no clutter)
- Readable text
- Good lighting
- Meaningful data shown

### ❌ Avoid
- Empty screens
- Personal info visible
- Blurry images
- Watermarks
- Test data like "Test User"
